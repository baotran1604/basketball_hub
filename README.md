# Basketball Events & Teams Platform

## Description
This is a web platform for the basketball community where users can create an account, find or create teams and events, join tryouts, and register for tournaments. Team managers can post tryouts and event organizers can publish tournaments to reach more players. The application is built with Python and Flask according to the course technical requirements.

## Key features
- User registration and login (different roles: Player, Team Manager, Event Organizer)
- Browse and search teams and upcoming events/tournaments by location, date or skill level
- Create your own team or event with details (name, description, location, contact info)
- Team managers can post tryouts with date, time, location, requirements and open spots
- Event organizers can post tournaments with schedule, registration link, rules and prizes
- Apply to join a team or register for tryouts/tournaments using HTML forms
- All pages are server-rendered HTML with clean, modern custom CSS styling
- Responsive design that works on both desktop and mobile devices (pure CSS)

## Technologies
- Backend: Python 3.10 + Flask
- Database: SQLite 
- Frontend: HTML templates (Jinja2) + custom CSS
- Authentication: Flask sessions
- Version control: Git + GitHub
